## About

Spring PetClinic is a sample application designed to show how a simple but non-trivial application can be built using the Spring ecosystem. It demonstrates:

- Spring Boot 4.0.1 auto-configuration
- Spring MVC for web layer
- Spring Data JPA for data access
- Thymeleaf for server-side HTML rendering
- Bean Validation (Jakarta Validation)
- JCache-based caching with Caffeine
- Internationalization (i18n) support
- Spring Boot Actuator for monitoring

## Quick Start

```bash
git clone https://github.com/awatson88/spring-petclinic.git
cd spring-petclinic
./mvnw spring-boot:run
```

Then open http://localhost:8080/ in your browser.
